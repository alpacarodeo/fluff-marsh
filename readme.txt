Download autohotkey (Windows only) here: https://www.autohotkey.com/

- Replace "pic.png" with your own image (I use a low res 500x300 size image for quick uploads) 
- Double click "approval.ahk" to run the script
- Navigate to freshly approved post
- Make sure the "Comment" image and "Write a comment..." text box are both visible on the screen
- Ctrl + Shift + V to automatically paste the image into the comment box
- Press enter when image is uploaded to finish posting comment
